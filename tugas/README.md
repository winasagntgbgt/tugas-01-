### 📌 1. Penjelasan Konsep Array

Array dalam Python disebut list.
List digunakan untuk menyimpan banyak data dalam satu variabel.

💻Contoh:

nilai = [80, 75, 90, 60]

💡Penjelasan:
- nilai adalah tempat menyimpan data

- setiap data bisa diakses dengan nomor urutan (indeks)

- Contoh: nilai[0] berarti data pertama

💡Pada program ini, array digunakan untuk:
- Menyimpan 10 nilai mahasiswa

- Mengolah data seperti mencari nilai tertinggi, terendah, dan rata-rata

### 📊 3. Analisis Kompleksitas
| No | ⚙️ Proses                | 📝 Penjelasan                                   | ⏱️ Kompleksitas |
|----|--------------------------|-----------------------------------------------|----------------|
| 1  | 🧮 Input nilai           | Memasukkan n data ke dalam array              | O(n)           |
| 2  | 🔝 Nilai tertinggi       | Mengecek semua elemen dalam array             | O(n)           |
| 3  | 🔻 Nilai terendah        | Mengecek semua elemen dalam array             | O(n)           |
| 4  | 📊 Rata-rata             | Menjumlahkan semua nilai lalu dibagi n        | O(n)           |
| 5  | ✅ Jumlah lulus          | Mengecek setiap nilai (>= 60)                 | O(n)           |
| 6  | 📈 Grafik                | Mengolah seluruh data untuk visualisasi       | O(n)           |

## 📈 Kesimpulan
⚡Semua proses menggunakan perulangan (loop)

⚡Kompleksitas keseluruhan adalah O(n)

⚡Artinya: semakin banyak data, waktu proses bertambah secara linear
